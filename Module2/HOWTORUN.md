# Instruction to Run Module 2 Assginment Script

Step 1: Upload the folder Module2 on your local Jupyter Lab
Step 2: In the code cell for step 1, change the directory of 'images' that is used for the calibration with the directory on your local machine
Step 3: In the code cell for step 3, change the directory of 'image_path' that is used for the validation experiment with the directory on your local machine
Step 4: Run all the code cells sequentially from top to bottom

* Notes: You can check the pre-runned ouput result in the folder 'module2_outputs'
